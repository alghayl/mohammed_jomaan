Introduction

A website consisting of seven linked pages has been developed to showcase personal information, skills, and projects, in addition to providing login and account creation functionality for users.

1. Project Objectives

  • Create a complete personal website with multiple pages.
  • Present personal information and skills in an organized way.
  • Provide a section to showcase previous projects.
  • Add a contact section for communication.
  • Provide a login and account creation system.

2. Page Description

  • Home Page: A simple welcome interface with name, image, and navigation bar.
  • About Me: Contains personal information about the website owner.
  • Skills: Displays technical skills with a neat design.
  • Projects: A page to showcase previous projects in an organized way.
  • Contact Me: A form to communicate via email.
  • Login: User login with email and password.
  • Create Account: New user registration with basic details.

3. Tools and Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • Flexbox/Grid
  • Responsive Design

4. Challenges

The main challenges included making the design responsive across different devices, designing secure and user-friendly login and registration forms, and organizing projects and skills in a clean and professional way.

5. Results

  • A personal website with 7 interconnected pages.
  • A responsive and attractive design.
  • Login and account creation functionality.
  • Easy navigation between pages.

6. Conclusion

The project represents a practical application of basic web development concepts. It can be further enhanced in the future by connecting login and registration forms to an actual database (MySQL or MongoDB) with a server-side programming language (such as PHP or Node.js).
